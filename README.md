# Modular Text Adventure Engine

A Java text-based adventure game engine originally created for a programming 
course and now being redesigned as a modular portfolio project.

Current playable game: Alice in Wonderland.

## Features
- CSV-driven locations, items, characters and actions
- Inventory system
- Expandable architecture for multiple future games

## How to Run
1. Clone the repository
2. Open the project folder
3. Compile:
	```
	javac *.java
	```
4. Run:
	```
	java Launcher Alice
	```
