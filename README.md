# Modular Text Adventure Engine

A Java text-based adventure game engine originally created for a programming 
course and now being redesigned as a modular portfolio project.

Current playable game: Alice in Wonderland.

## Features
- Modular game folders
- CSV-driven locations, items, characters, encounters and actions
- Inventory system
- Encounter selection based on player state, tokens, items, and location
- Expandable architecture for multiple games

## Why I Built This
Originally created as a course project, then redesigned into a reusable engine 
to practice object-oriented design, file-driven architecture, and modular game 
systems.

## How to Run
1. Clone the repository
2. Open the project folder
3. Compile:
	```
	javac src/*.java
	```
4. Run:
	```
	java -cp src Launcher
	```
